# Step 05: Produção Visual

O Davi Design materializa as artes com base no formato definido no Menu Inicial.

## Context Loading
- `squads/instagram-autonomo-nyx/output/content-draft.md` — Roteiro visual do Fábio.
- `squads/instagram-autonomo-nyx/output/menu-inicial.md` — Formato (Carrossel ou Post Único).
- `squads/instagram-autonomo-nyx/pipeline/data/quality-criteria.md` — Regras de design.

## Instructions

### Se formato = Carrossel:
1. Gere uma arte para CADA slide descrito no content-draft (8-10 imagens).
2. Nomeie os arquivos: `slide-01.png`, `slide-02.png`, ..., `slide-10.png`.
3. Use a ferramenta generate_image para cada slide.
4. Mantenha consistência visual entre todos os slides (mesma paleta, tipografia, estilo).
5. Salve todos em `squads/instagram-autonomo-nyx/output/images/`.

### Se formato = Post Único:
1. Gere 1 arte de impacto máximo para o post.
2. Nomeie: `post-unico.png`.
3. Salve em `squads/instagram-autonomo-nyx/output/images/`.

## Quality Rules
- Proporção: 1080x1350 (3:4 portrait).
- Paleta: Dark Mode + Neon (Púrpura #A020F0 / Ciano #00FFFF).
- Estilo: Glassmorphism holográfico, bordas neon finas.
- Títulos em Português sobre a arte (OBRIGATÓRIO).
- Logo NYX no slide final (Carrossel) ou canto inferior (Post Único).
