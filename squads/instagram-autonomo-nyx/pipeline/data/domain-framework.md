# Domain Framework: Instagram Content Architecture (NYX)

## 🔄 Processo Operacional (Step-by-Step)

1. **Fase de Input (Curadoria):**
   - Identificar 3 pautas quentes em IA/Design.
   - Filtrar pela "Utilidade Imediata" (O que o usuário pode usar agora?).
   - Resultado: 1 pauta selecionada com fonte e resumo.

2. **Fase de Ideação (Ângulos):**
   - Gerar 5 perspectivas baseadas na pauta (Medo, Oportunidade, Educacional, Contrário, Inspiracional).
   - Validar se o ângulo permite um CTA de "Comente IA".
   - Resultado: 5 opções de ângulos para o usuário.

3. **Fase de Scripting (Copywriting):**
   - **Feed**: Estratégia de Carrossel (Hook -> Problema -> Solução -> CTA).
   - **Reels**: Estratégia de Loop (Hook 1.5s -> Explicação visual -> Chamada).
   - Resultado: Legenda + Roteiro Visual.

4. **Fase de Produção (Design):**
   - Criar prompt para IA Generativa baseado no tema.
   - Aplicar template de marca no Canva (Cores NYX, Logo, Tipografia).
   - Resultado: Imagem final ou Mockup.

5. **Fase de Veredito (Review):**
   - Checar conformidade com o tom NYX.
   - Validar links e CTAs.
   - Resultado: APROVADO ou REJEITADO (com feedback).
